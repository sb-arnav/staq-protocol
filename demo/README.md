# Demo: Credential-Gated DeFi

A simple demo showing how a DeFi lending app can read Staq credentials and offer better rates to verified users.

## The idea

A lending page that:
1. Connects your wallet
2. Reads your Staq credentials via the SDK
3. Calculates your Staq Score
4. Shows different collateral requirements based on your score

**Score 0 (no credentials):** 150% collateral required
**Score 300+ (some credentials):** 120% collateral
**Score 600+ (many credentials):** 100% collateral
**Score 800+ (expert):** 80% collateral

This demonstrates the core value prop: verified knowledge = better financial access.
