# Staq Credential Program (Anchor)

Federico's Anchor program goes here.

## Instructions

1. `register_issuer` — Protocol admin approves a new issuer
2. `register_credential` — Approved issuer writes a credential to a user's wallet
3. `verify_credential` — Any program derives the PDA and reads the account
4. `deactivate_issuer` — Protocol admin removes an issuer
5. `stake` — User locks Flux before a module attempt
6. `resolve_pass` — Authority returns Flux after passing
7. `resolve_fail` — Authority burns Flux after failing
